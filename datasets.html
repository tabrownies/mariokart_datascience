These are the datasets we use for our analysis. Find the code to create them at this github. Each is a parquet file you can parse with Pandas.

Mario Kart 8 Deluxe World Record Times

This dataset contains the information for each world world record for Mario Kart 8 Deluxe. These are found on mkwrs.com.

Download

Mario Kart 8 Deluxe Character Statistics

This dataset contains the statistics for each character in Mario Kart 8 Deluxe. This includes speed, acceleration, traction, handling, and mini-turbo.

Download

Mario Kart 8 Deluxe Kart Statistics

This dataset contains the statistics for each kart in Mario Kart 8 Deluxe. This includes speed, acceleration, traction, handling, and mini-turbo.

Download

Mario Kart 8 Deluxe Tire Statistics

This dataset contains the statistics for each set of tires in Mario Kart 8 Deluxe. This includes speed, acceleration, traction, handling, and mini-turbo.

Download

Mario Kart 8 Deluxe Glider Statistics

This dataset contains the statistics for each glider type in Mario Kart 8 Deluxe. This includes speed, acceleration, traction, handling, and mini-turbo.
